# Anexos

Esta carpeta contiene material complementario y anexos de la tesis del proyecto educativo.

## Contenido de los Anexos

### Anexo A: Documentación Administrativa
- Autorizaciones
- Permisos
- Documentos institucionales

### Anexo B: Instrumentos de Investigación
- Cuestionarios utilizados
- Guiones de entrevistas
- Protocolos de observación
- Escalas de valoración

### Anexo C: Datos Brutos
- Tablas de datos
- Transcripciones de entrevistas
- Resultados de cuestionarios

### Anexo D: Material Fotográfico
- Fotografías del desarrollo del proyecto
- Imágenes de actividades
- Evidencias visuales

### Anexo E: Producciones de los Estudiantes
- Ejemplos de trabajos realizados
- Muestras de actividades
- Evidencias de aprendizaje

### Anexo F: Material Adicional
- Recursos complementarios
- Enlaces de interés
- Documentación extra

## Organización

Crear carpetas específicas para cada anexo:

```
anexos/
├── anexo_A_documentacion/
├── anexo_B_instrumentos/
├── anexo_C_datos/
├── anexo_D_fotografias/
├── anexo_E_producciones/
└── anexo_F_adicional/
```

## Formato de Archivos

- **Documentos**: PDF (preferiblemente)
- **Imágenes**: JPG, PNG
- **Datos**: XLS, CSV
- **Audio/Video**: MP3, MP4

## Consideraciones de Privacidad

- Anonimizar datos personales de estudiantes
- Obtener permisos para uso de imágenes
- Proteger información sensible
- Cumplir con normativa de protección de datos (RGPD)

## Referencias en la Tesis

Al citar anexos en la tesis, usar el formato:
- "(Ver Anexo A)"
- "Como se muestra en el Anexo B..."
- "Los resultados completos se encuentran en el Anexo C"
