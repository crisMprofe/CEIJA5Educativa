# CEIJA5 Educativa - Plataforma de Gestión de Tesis

Bienvenido a la plataforma educativa CEIJA5 para la gestión y desarrollo de proyectos de tesis académicas.

## 📚 Descripción

CEIJA5 Educativa es una plataforma web diseñada para facilitar la gestión, seguimiento y desarrollo de proyectos de tesis en entornos educativos. Proporciona herramientas y recursos para estudiantes, tutores y coordinadores académicos.

## ✨ Características

- **Gestión de Proyectos**: Crear, editar y dar seguimiento a proyectos de tesis
- **Recursos Educativos**: Guías metodológicas, plantillas y material de apoyo
- **Sistema de Estados**: Seguimiento del progreso (Activo, En Revisión, Completado)
- **Interfaz Intuitiva**: Diseño responsive y fácil de usar
- **Documentación Completa**: Guías detalladas y plantillas estándar

## 🚀 Inicio Rápido

### Requisitos Previos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- No requiere instalación de dependencias adicionales

### Instalación

1. Clone el repositorio:
```bash
git clone https://github.com/crisMprofe/CEIJA5Educativa.git
```

2. Navegue al directorio del proyecto:
```bash
cd CEIJA5Educativa
```

3. Abra el archivo `index.html` en su navegador:
```bash
# En Windows
start index.html

# En macOS
open index.html

# En Linux
xdg-open index.html
```

## 💡 Uso

### Crear un Nuevo Proyecto

1. Haga clic en el botón "Iniciar Proyecto" en la página principal
2. Complete el formulario con la información del proyecto:
   - Título del proyecto
   - Nombre del autor
   - Descripción
   - Fecha de inicio
   - Estado del proyecto
3. Haga clic en "Crear Proyecto"

### Visualizar Proyectos

Los proyectos se muestran en la sección "Proyectos de Tesis" con tarjetas que incluyen:
- Título y autor
- Descripción breve
- Fecha
- Estado actual (código de colores)

### Acceder a Recursos

La sección "Recursos Educativos" proporciona:
- **Guías de Investigación**: Metodología y técnicas académicas
- **Plantillas**: Formatos estándar para documentos
- **Bibliografía**: Referencias recomendadas

## 🎨 Personalización

### Modificar Estilos

Edite el archivo `styles/main.css` para personalizar:

- Fuentes y tipografía
- Espaciado y diseño
- Efectos de animación

### Agregar Funcionalidades

Edite el archivo `scripts/main.js` para:
- Agregar nuevos campos a los proyectos
- Implementar filtros y búsqueda
- Integrar con bases de datos
- Agregar validaciones personalizadas
