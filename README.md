# CEIJA5Educativa

## Proyecto Educativo - Tesis

Bienvenido al repositorio del proyecto educativo CEIJA5. Este repositorio contiene la documentación completa de la tesis del proyecto educativo, así como los recursos y materiales asociados.

## Estructura del Proyecto

```
CEIJA5Educativa/
├── docs/
│   └── tesis/              # Documentación de la tesis
│       ├── capitulos/      # Capítulos de la tesis
│       ├── imagenes/       # Imágenes y gráficos
│       └── referencias/    # Referencias bibliográficas
├── recursos/
│   ├── actividades/        # Actividades educativas
│   ├── materiales/         # Materiales didácticos
│   └── evaluaciones/       # Instrumentos de evaluación
└── anexos/                 # Anexos y material complementario
```

## Contenido de la Tesis

La tesis del proyecto educativo incluye los siguientes componentes:

1. **Resumen y Abstract**
2. **Introducción**
3. **Marco Teórico**
4. **Metodología**
5. **Desarrollo del Proyecto**
6. **Resultados y Análisis**
7. **Conclusiones**
8. **Referencias Bibliográficas**
9. **Anexos**

## Recursos Educativos

Este repositorio también contiene recursos educativos organizados en:

- **Actividades**: Propuestas de actividades para el desarrollo del proyecto
- **Materiales**: Material didáctico y de apoyo
- **Evaluaciones**: Instrumentos de evaluación y seguimiento

## Cómo Usar Este Repositorio

1. Navega a la carpeta `docs/tesis/` para acceder a los capítulos de la tesis
2. Consulta la carpeta `recursos/` para los materiales educativos
3. Revisa los `anexos/` para información complementaria
