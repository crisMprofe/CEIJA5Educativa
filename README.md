## Configuración de GitHub Copilot Chat / GitHub Copilot Chat Setup

### Español

Si recibes el mensaje "Chat took too long to get ready. Please ensure you are signed in to GitHub and that the extension GitHub.copilot-chat is installed and enabled", sigue estos pasos:

#### 1. Iniciar Sesión en GitHub

1. Abre Visual Studio Code
2. Haz clic en el icono de cuenta en la esquina inferior izquierda
3. Selecciona "Sign in to GitHub"
4. Se abrirá una ventana del navegador para autorizar VS Code
5. Inicia sesión con tus credenciales de GitHub
6. Autoriza la aplicación cuando se te solicite

#### 2. Instalar la Extensión GitHub Copilot Chat

1. Abre VS Code
2. Haz clic en el icono de Extensiones en la barra lateral izquierda (o presiona `Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Busca "GitHub Copilot Chat"
4. Haz clic en "Install" en la extensión "GitHub Copilot Chat"
5. Espera a que se complete la instalación

#### 3. Habilitar la Extensión

1. Una vez instalada, la extensión debería habilitarse automáticamente
2. Si no está habilitada:
   - Ve a Extensiones (`Ctrl+Shift+X` / `Cmd+Shift+X`)
   - Busca "GitHub Copilot Chat"
   - Si ves un botón "Enable", haz clic en él
   - Si ves "Disable", significa que ya está habilitada

#### 4. Verificar la Configuración

1. Busca el icono de chat en la barra lateral (icono de chat/conversación)
2. Haz clic en él para abrir el panel de GitHub Copilot Chat
3. Si todo está configurado correctamente, deberías poder escribir mensajes

#### Solución de Problemas

- **Asegúrate de tener una suscripción activa de GitHub Copilot**
- Reinicia VS Code después de instalar la extensión
- Verifica que estés conectado a Internet
- Comprueba que tu cuenta de GitHub tenga acceso a GitHub Copilot
- Si el problema persiste, intenta:
  - Cerrar sesión y volver a iniciar sesión
  - Desinstalar y reinstalar la extensión
  - Actualizar VS Code a la última versión

### English

If you receive the message "Chat took too long to get ready. Please ensure you are signed in to GitHub and that the extension GitHub.copilot-chat is installed and enabled", follow these steps:

#### 1. Sign In to GitHub

1. Open Visual Studio Code
2. Click on the account icon in the bottom-left corner
3. Select "Sign in to GitHub"
4. A browser window will open to authorize VS Code
5. Sign in with your GitHub credentials
6. Authorize the application when prompted

#### 2. Install the GitHub Copilot Chat Extension

1. Open VS Code
2. Click on the Extensions icon in the left sidebar (or press `Ctrl+Shift+X` / `Cmd+Shift+X`)
3. Search for "GitHub Copilot Chat"
4. Click "Install" on the "GitHub Copilot Chat" extension
5. Wait for the installation to complete

#### 3. Enable the Extension

1. Once installed, the extension should be enabled automatically
2. If it's not enabled:
   - Go to Extensions (`Ctrl+Shift+X` / `Cmd+Shift+X`)
   - Search for "GitHub Copilot Chat"
   - If you see an "Enable" button, click it
   - If you see "Disable", it means it's already enabled

#### 4. Verify the Setup

1. Look for the chat icon in the sidebar (chat/conversation icon)
2. Click on it to open the GitHub Copilot Chat panel
3. If everything is set up correctly, you should be able to type messages

#### Troubleshooting

- **Make sure you have an active GitHub Copilot subscription**
- Restart VS Code after installing the extension
- Verify that you're connected to the Internet
- Check that your GitHub account has access to GitHub Copilot
- If the problem persists, try:
  - Signing out and signing back in
  - Uninstalling and reinstalling the extension
  - Updating VS Code to the latest version
