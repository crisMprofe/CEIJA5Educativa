# Referencias Bibliográficas

Esta carpeta contiene material bibliográfico y documentos de referencia utilizados en la elaboración de la tesis.

## Contenido

- Artículos científicos en PDF
- Extractos de libros relevantes
- Normativa educativa
- Informes y documentos oficiales
- Material de referencia consultado

## Organización Sugerida

```
referencias/
├── articulos/
│   ├── pedagogia/
│   └── metodologia/
├── libros/
├── normativa/
└── otros/
```

## Gestión de Referencias

Se recomienda utilizar un gestor de referencias bibliográficas como:
- Zotero
- Mendeley
- EndNote

## Formato de Citación

La tesis utiliza el formato APA (7ª edición). Consultar el archivo `07_referencias.md` para ver el formato completo de citación.

## Notas

- Mantener una copia digital de todos los documentos citados
- Organizar las referencias por temática o capítulo
- Actualizar regularmente la lista de referencias en `07_referencias.md`
