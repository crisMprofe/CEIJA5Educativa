# Capítulos de la Tesis

Esta carpeta está destinada a almacenar versiones alternativas, borradores o capítulos adicionales de la tesis.

## Contenido Sugerido

- Borradores de capítulos en proceso de revisión
- Versiones anteriores de capítulos ya finalizados
- Capítulos adicionales o anexos extensos
- Material complementario para los capítulos principales

## Organización

Se recomienda organizar los archivos con la siguiente nomenclatura:

```
capitulo_01_borrador_v1.md
capitulo_01_borrador_v2.md
capitulo_02_version_extendida.md
```

## Notas

Los capítulos principales de la tesis se encuentran en el directorio superior (`docs/tesis/`).
