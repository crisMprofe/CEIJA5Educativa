# Imágenes y Gráficos

Esta carpeta contiene todas las imágenes, gráficos, diagramas y tablas utilizados en la tesis.

## Organización

Se recomienda organizar las imágenes por capítulo:

```
imagenes/
├── capitulo_01/
│   ├── figura_1_1.png
│   └── figura_1_2.png
├── capitulo_02/
│   ├── grafico_2_1.png
│   └── tabla_2_1.png
└── capitulo_03/
    └── diagrama_3_1.png
```

## Formato de Archivos

- **Imágenes**: PNG, JPG (alta resolución, mínimo 300 dpi)
- **Gráficos**: PNG, SVG (preferiblemente vectoriales)
- **Diagramas**: PNG, SVG, PDF

## Nomenclatura

Usar nombres descriptivos que incluyan:
- Tipo de elemento (figura, gráfico, tabla, diagrama)
- Número de capítulo
- Número correlativo
- Breve descripción

Ejemplo: `figura_4_1_cronograma_actividades.png`

## Citas

Al insertar imágenes en la tesis, recordar incluir:
- Número de figura/tabla
- Título descriptivo
- Fuente (si no es de elaboración propia)
