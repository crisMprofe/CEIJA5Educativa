# Guía del Usuario - CEIJA5 Educativa

## Introducción

Esta guía proporciona instrucciones paso a paso para utilizar la plataforma CEIJA5 Educativa.

## Navegación

### Menú Principal

La barra de navegación superior contiene los siguientes enlaces:

- **Inicio**: Página principal con información general
- **Proyectos**: Listado de proyectos de tesis
- **Recursos**: Material educativo y guías
- **Contacto**: Formulario para comunicarse con el equipo

## Gestión de Proyectos

### Crear un Nuevo Proyecto

1. **Acceder al formulario**
   - Haga clic en el botón "Iniciar Proyecto" en la página de inicio
   - Se abrirá un formulario modal

2. **Completar información**
   - **Título**: Ingrese el título completo de su proyecto
   - **Autor**: Su nombre completo
   - **Descripción**: Breve resumen del proyecto (2-3 líneas)
   - **Fecha**: Seleccione la fecha de inicio
   - **Estado**: Elija el estado actual (Activo, En Revisión, Completado)

3. **Guardar**
   - Haga clic en "Crear Proyecto"
   - Verá una confirmación de éxito
   - El proyecto aparecerá en la lista automáticamente

### Visualizar Proyectos

En la sección "Proyectos de Tesis" encontrará:

- **Tarjetas de proyecto**: Cada proyecto se muestra en una tarjeta individual
- **Información visible**: Título, autor, descripción, fecha y estado
- **Indicador de estado**: 
  - Verde = Activo
  - Amarillo = En Revisión
  - Azul = Completado

## Solución de Problemas

### El formulario no se muestra

- Verifique que JavaScript esté habilitado en su navegador
- Refresque la página (F5)
- Limpie la caché del navegador

### Los proyectos no aparecen

- Asegúrese de haber creado al menos un proyecto
- Verifique la consola del navegador para errores (F12)

### Problemas de visualización

- Actualice su navegador a la última versión
- Pruebe con un navegador diferente
- Verifique que los archivos CSS estén cargados correctamente

## Preguntas Frecuentes

**P: ¿Los datos se guardan permanentemente?**  
R: Actualmente los datos son temporales. Una futura versión incluirá persistencia de datos.

**P: ¿Puedo exportar mis proyectos?**  
R: Esta funcionalidad está en desarrollo.

**P: ¿Es necesario crear una cuenta?**  
R: No por el momento. El sistema es de acceso libre.

**P: ¿Puedo trabajar offline?**  
R: Sí, una vez cargada la página, puede trabajar sin conexión.
