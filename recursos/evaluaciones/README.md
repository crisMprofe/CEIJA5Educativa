# Instrumentos de Evaluación

Esta carpeta contiene los instrumentos y herramientas de evaluación utilizados en el proyecto.

## Tipos de Evaluación

### Evaluación Inicial (Diagnóstica)
- Pruebas de conocimientos previos
- Cuestionarios iniciales
- Observación diagnóstica

### Evaluación Formativa (Procesual)
- Rúbricas de evaluación
- Listas de control (checklists)
- Diarios de aprendizaje
- Portafolios
- Observación sistemática

### Evaluación Sumativa (Final)
- Pruebas escritas
- Proyectos finales
- Presentaciones orales
- Exámenes

### Autoevaluación y Coevaluación
- Fichas de autoevaluación
- Instrumentos de coevaluación entre pares
- Reflexiones metacognitivas

## Instrumentos Incluidos

### Rúbricas
Matrices de evaluación con criterios y niveles de desempeño

### Listas de Cotejo
Listas de verificación para observar conductas o cumplimiento de criterios

### Escalas de Valoración
Instrumentos para medir grado de consecución de objetivos

### Registros de Observación
Plantillas para registrar observaciones sistemáticas

## Plantilla de Rúbrica

```markdown
# Rúbrica: [Nombre de la Actividad/Proyecto]

| Criterio | Excelente (4) | Bueno (3) | Aceptable (2) | Insuficiente (1) |
|----------|---------------|-----------|---------------|------------------|
| Criterio 1 | Descripción | Descripción | Descripción | Descripción |
| Criterio 2 | Descripción | Descripción | Descripción | Descripción |
| Criterio 3 | Descripción | Descripción | Descripción | Descripción |
```

## Consideraciones

- Los instrumentos deben estar alineados con los objetivos de aprendizaje
- Deben ser claros, objetivos y comprensibles
- Proporcionar retroalimentación constructiva
- Facilitar el seguimiento del progreso del estudiante
