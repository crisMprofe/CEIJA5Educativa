# Actividades Educativas

Esta carpeta contiene las propuestas de actividades educativas diseñadas para el proyecto.

## Tipos de Actividades

### Actividades de Inicio
- Actividades de motivación
- Diagnóstico de conocimientos previos
- Introducción al tema

### Actividades de Desarrollo
- Actividades prácticas
- Trabajo en grupo
- Investigación y exploración
- Proyectos colaborativos

### Actividades de Cierre
- Actividades de consolidación
- Presentaciones y exposiciones
- Evaluación de aprendizajes

## Estructura de una Actividad

Cada actividad debe incluir:

1. **Título**: Nombre descriptivo de la actividad
2. **Objetivos**: Qué se pretende conseguir
3. **Duración**: Tiempo estimado
4. **Materiales**: Recursos necesarios
5. **Desarrollo**: Paso a paso de la actividad
6. **Evaluación**: Criterios de evaluación
7. **Adaptaciones**: Posibles adaptaciones para diferentes niveles

## Ejemplo de Plantilla

Crear archivos con el formato: `actividad_XX_nombre.md`

```markdown
# Actividad [Número]: [Nombre]

## Objetivos
- Objetivo 1
- Objetivo 2

## Duración
[X] minutos/horas

## Materiales
- Material 1
- Material 2

## Desarrollo
1. Paso 1
2. Paso 2
3. Paso 3

## Evaluación
[Criterios de evaluación]

## Adaptaciones
[Posibles adaptaciones]
```
