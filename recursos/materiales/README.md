# Materiales Didácticos

Esta carpeta contiene los materiales didácticos y de apoyo utilizados en el proyecto educativo.

## Tipos de Materiales

### Materiales Impresos
- Fichas de trabajo
- Cuadernillos
- Guías didácticas
- Hojas de ejercicios

### Materiales Digitales
- Presentaciones (PowerPoint, Google Slides)
- Videos educativos
- Recursos interactivos
- Enlaces a recursos web

### Materiales Manipulativos
- Juegos educativos
- Material concreto
- Recursos para experimentos

## Organización

```
materiales/
├── fichas/
├── presentaciones/
├── videos/
├── interactivos/
└── otros/
```

## Formato de Archivos

- **Documentos**: PDF, DOCX, ODT
- **Presentaciones**: PPTX, PDF
- **Multimedia**: MP4, MP3, PNG, JPG
- **Interactivos**: HTML, Enlaces a plataformas web

## Licencias y Atribuciones

Si se utilizan materiales de terceros, incluir:
- Autor original
- Licencia de uso
- Enlace a la fuente original

## Accesibilidad

Asegurar que los materiales sean accesibles:
- Textos legibles (tamaño de fuente adecuado)
- Contraste de colores
- Alternativas textuales para imágenes
- Formatos compatibles con lectores de pantalla
